# 时序图片播放器

一个基于Flask的时序图片播放器，支持48张图片的顺序播放和时间进度条控制。

## 功能特点
- 48张顺序图片滚动播放
- 交互式时间进度条
- 播放速度控制
- 缩略图导航
- 响应式设计
